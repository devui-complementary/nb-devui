### d-smart-input 参数
|  参数   |   类型    | 默认 | 说明                              |
| :-----: | :-------: | :--: | :------------------------------- |
|clearable| `boolean` | false| 【可选】是否可清空                 |
| onClear |  `func`   |  --  | 【可选】点击清除按钮触发的函数      |
|  size   | `string`  |  ""  | 【可选】输入框尺寸,有三种选择'lg','','sm' |
| placeholder |  `string`   |  ""  | 【可选】允许用户自定义placeholder  |
|  disabled   |  `string`   |  false  | 【可选】是否禁用 input 组件  |
